# 网页版天气查询系统
## 实现功能：
- 输入城市名，获取该城市最新天气情况
- 点击「帮助」，获取帮助信息
- 点击「历史」，获取历史查询信息

## 涉及知识
- flask
- html
  - form
- css,使用网页模板渲染
- database
- bootstrap
